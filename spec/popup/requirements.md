# 팝업 (Popup) 기능 요구사항

## 목적
확장 프로그램 팝업에서 탭 저장뿐 아니라, 저장된 탭 그룹을 빠르게 확인하고 탭을 직접 열 수 있는 기능을 제공한다.

## 기능 요구사항

| ID | 요구사항 |
|----|---------|
| PP-01 | 현재 탭 저장 버튼을 제공한다 |
| PP-02 | 모든 탭 저장(유지 / 닫기) 버튼을 제공한다 |
| PP-03 | 랜딩 페이지(Tab Archive)로 이동하는 버튼을 제공한다 |
| PP-04 | 저장된 탭 그룹 목록을 팝업 내에서 확인할 수 있다 |
| PP-05 | 목록의 탭 항목 클릭 시 해당 URL로 이동(탭 열기)할 수 있다 |
| PP-06 | 저장된 그룹이 없을 경우 안내 메시지를 표시한다 |
| PP-07 | 저장된 탭 목록을 접거나 펼칠 수 있다 (Fold/Unfold) |
| PP-08 | Fold된 그룹 카드는 접혀 있음을 시각적으로 알 수 있는 표시를 제공한다 |

## 비기능 요구사항
- 팝업 크기 제한(320~400px 폭)에 맞게 레이아웃을 설계한다
- 팝업 열기 시 storage에서 최신 데이터를 읽어 렌더링한다

## 범위 밖 (Out of Scope)
- 팝업 내 탭 삭제 / 그룹 관리 기능 (랜딩 페이지에서 담당)
- 팝업 내 검색 기능

## 수용 기준 (Acceptance Criteria)
- 팝업을 열면 저장된 탭 그룹과 각 그룹의 탭 목록이 표시된다
- 탭 항목 클릭 시 해당 URL이 열리고 팝업이 닫힌다
- 저장된 그룹이 없으면 "저장된 탭이 없습니다" 안내 메시지가 표시된다
