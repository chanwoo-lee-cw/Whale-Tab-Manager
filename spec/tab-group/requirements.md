# 탭 그룹 (Tab Group) 기능 요구사항

## 목적
저장된 탭들을 그룹 단위로 묶어 관리할 수 있도록 한다.
그룹은 탭 저장 시 자동 생성되며, 사용자가 이름을 변경하거나 삭제할 수 있다.

## 기능 요구사항

| ID | 요구사항 |
|----|---------|
| TG-01 | 탭 저장 시 저장된 탭들이 자동으로 하나의 그룹으로 묶인다 |
| TG-02 | 사용자는 그룹 이름을 변경할 수 있다 |
| TG-03 | 사용자는 그룹 전체를 삭제할 수 있다 |
| TG-04 | 사용자는 그룹 내 특정 탭을 삭제할 수 있다 |
| TG-05 | 그룹은 생성 시각 기준 내림차순(최신순)으로 표시된다 |
| TG-06 | 그룹 내 탭 수가 표시된다 |

## 비기능 요구사항
- 그룹 및 탭 데이터는 `chrome.storage.local`에 영속적으로 저장된다
- 그룹 삭제/수정은 storage에 즉시 반영된다

## 범위 밖 (Out of Scope)
- 그룹 간 탭 이동
- 그룹 순서 수동 변경 (드래그 앤 드롭)
- 그룹 내보내기/가져오기

## 수용 기준 (Acceptance Criteria)
- 탭 저장 시 랜딩 페이지에 새 그룹이 나타난다
- 그룹 이름 클릭/편집 후 변경 내용이 storage에 저장된다
- 그룹 삭제 시 해당 그룹과 소속 탭이 모두 제거된다
- 그룹 내 탭 삭제 시 해당 탭만 제거되고 그룹은 유지된다
