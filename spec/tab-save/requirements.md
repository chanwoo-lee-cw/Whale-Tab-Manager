# 탭 저장 (Tab Save) 기능 요구사항

## 목적
사용자가 현재 열려 있는 탭(단일 또는 전체)을 저장하여 나중에 다시 열 수 있도록 한다.

## 기능 요구사항

| ID | 요구사항 |
|----|---------|
| TS-01 | 사용자는 현재 활성 탭을 저장할 수 있다 |
| TS-02 | 사용자는 현재 창의 모든 탭을 한 번에 저장할 수 있다 |
| TS-03 | 모든 탭 저장 시 저장 후 탭을 모두 닫을지 선택할 수 있다 (저장 후 닫기) |
| TS-04 | 모든 탭 저장 시 탭을 닫지 않고 유지할 수 있다 (저장 후 유지) |
| TS-05 | 저장 시 탭의 title, url, favicon URL을 함께 저장한다 |
| TS-06 | 저장된 탭은 자동으로 하나의 그룹으로 묶인다 |
| TS-07 | 그룹 이름은 저장 시각을 기본값으로 한다 (예: `2026-03-25 14:30`) |
| TS-08 | 사용자는 탭 그룹에 TTL(Time To Live)을 지정할 수 있다 |
| TS-09 | TTL은 특정 시간(시간 단위) 또는 특정 날짜로 지정할 수 있다 |
| TS-10 | TTL이 만료된 탭 그룹은 자동으로 삭제된다 |
| TS-11 | TTL 만료 하루 전에 사용자에게 알림을 표시한다 |

## 비기능 요구사항
- 저장 작업은 사용자가 인지할 수 있는 딜레이 없이 즉시 완료되어야 한다
- `chrome.storage.local`을 사용하며 저장 데이터는 title, url, favIconUrl만 포함한다

## 범위 밖 (Out of Scope)
- 다른 창(window)의 탭 저장
- 탭 그룹(Chrome Tab Groups) 단위 저장
- 클라우드 동기화 (`chrome.storage.sync`)
- 개별 탭 단위 TTL (그룹 단위만 지원)

## 수용 기준 (Acceptance Criteria)
- 팝업에서 "현재 탭 저장" 버튼 클릭 시 활성 탭이 새 그룹으로 저장된다
- 팝업에서 "모든 탭 저장 후 닫기" 클릭 시 모든 탭이 저장되고 탭이 닫힌다
- 팝업에서 "모든 탭 저장 후 유지" 클릭 시 모든 탭이 저장되고 탭은 그대로 유지된다
- 저장 완료 후 랜딩 페이지 또는 팝업에서 저장된 그룹이 확인된다
- TTL을 시간 단위 또는 날짜로 설정할 수 있다
- TTL 만료 시 해당 그룹이 자동 삭제된다
- TTL 만료 하루 전 브라우저 알림이 표시된다
